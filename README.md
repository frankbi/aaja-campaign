# Frank Bi for AAJA President — campaign site

Live site: https://frankbi.github.io/aaja-campaign/

Everything lives in one file: `index.html`. No build step — edit it and the site updates within a minute or two of pushing.

## How to update content

Easiest way (no tools needed):

1. Open `index.html` on GitHub and click the pencil icon (Edit).
2. Make your change — the file is plain HTML and each section is labeled with comments like `<!-- EDIT ME -->`.
3. Click **Commit changes**. GitHub Pages republishes automatically in ~1 minute.
